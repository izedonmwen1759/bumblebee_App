      Date Dash @ Doblaz is an online RTC [Real-time Communication] dating platform for people all around the world to connect with one another. Simply fill in the details above and click the ok button. To get connected, please follow these steps;
      1. Click on the other user B profile
      2. A notification will be sent to user B to accept your request. Upon acceptance, a calendar will be shown
      on their portal for them to schedule a date and time for a face-time communication [Zoom].
      3. Once the date has been entered and sent by use B, user A feed will be updated [pay attention to the video icon].
      4. User A is required to pay a fee before setting up and sending the meeting Id to other users.
      * NOTE: You are required to setup a meeting based on the date choosen by user B.
      5. After payment has been made, create a Zoom meeting Id and obtain the code for the meeting inline with the calendar sent by user B.
      6. Click on the profile icon and select "Appointment", a dialogue feed will display then insert your meeting Id code into the text box and send to the receiver[user B]. User B will automatically get notified.
